# RESTful API Project

This project focuses on the fundamentals of interacting with and developing RESTful APIs. It covers HTTP/HTTPS protocols, consuming APIs using Python's `requests` library, and handling data structures like JSON and CSV.

## Project Structure
- `task_02_requests.py`: Python script to fetch data from an API, print specific fields, and save data into a CSV file.
- `main_02_requests.py`: The entry point script to execute the API consumption tasks.
- `posts.csv`: The generated CSV file containing the fetched post data.

## Objectives
- Understand HTTP/HTTPS request and response structures.
- Use the `requests` library for API interaction.
- Parse and process JSON data.
- Convert and store data in CSV format.

## Requirements
- Python 3.9+
- `requests` library
