# SQL - More queries

## Description
This project is part of the Higher Level Programming curriculum at Holberton School. It focuses on advanced SQL concepts in MySQL, including user creation and privilege management, primary and foreign key constraints, table relationships (`JOIN`, `LEFT JOIN`), aggregate functions, subqueries, and database constraints.

## Learning Objectives
By completing this project, you are expected to be able to explain to anyone, without the help of Google:
* How to create a new MySQL user and manage permissions (`GRANT`, `REVOKE`, `SHOW GRANTS`)
* What is a `PRIMARY KEY` and a `FOREIGN KEY`
* How to use `NOT NULL` and `UNIQUE` constraints
* How to retrieve data from multiple tables using different types of `JOIN` statements (`INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`)
* How to group records and aggregate data using `GROUP BY` and `HAVING`
* How to subquery (`IN`, `NOT IN`, sub-queries in `WHERE`)

## Requirements
* Allowed editors: `vi`, `vim`, `emacs`, `VS Code`
* All files are compiled/executed on Ubuntu 20.04 LTS using `MySQL 8.0`
* All SQL files should end with a new line
* All SQL queries should have a comment just before the query
* All SQL keywords should be in uppercase (`SELECT`, `WHERE`, `CREATE`, etc.)
* A `README.md` file at the root of the project folder is mandatory
