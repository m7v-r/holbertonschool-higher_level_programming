======================================
How to use the text_indentation module
======================================

This module provides a function `text_indentation(text)` that indents text based on punctuation.

First, import the function:
    >>> text_indentation = __import__('5-text_indentation').text_indentation

Test normal case with standard punctuation:
    >>> text_indentation("Hello. World? This is: Python")
    Hello.
    <BLANKLINE>
    World?
    <BLANKLINE>
    This is:
    <BLANKLINE>
    Python

Test stripping spaces after punctuation:
    >>> text_indentation("Sentence one.      Sentence two?")
    Sentence one.
    <BLANKLINE>
    Sentence two?
    <BLANKLINE>

Test bad text types (TypeError):
    >>> text_indentation(123)
    Traceback (most recent call last):
    ...
    TypeError: text must be a string

    >>> text_indentation(None)
    Traceback (most recent call last):
    ...
    TypeError: text must be a string

Test missing argument altogether:
    >>> text_indentation()
    Traceback (most recent call last):
    ...
    TypeError: text_indentation() missing 1 required positional argument: 'text'
