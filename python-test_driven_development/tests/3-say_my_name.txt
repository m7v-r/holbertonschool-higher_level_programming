==================================
How to use the say_my_name module
==================================

This module provides a function `say_my_name(first_name, last_name="")` that prints a full name.

First, import the function:
    >>> say_my_name = __import__('3-say_my_name').say_my_name

Test normal case with first and last name:
    >>> say_my_name("John", "Smith")
    My name is John Smith

    >>> say_my_name("Walter", "White")
    My name is Walter White

Test using the default value for last_name (empty string):
    >>> say_my_name("Bob")
    My name is Bob
