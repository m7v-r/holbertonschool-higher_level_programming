=========================================
How to use the lazy_matrix_mul module
=========================================

This module provides a function `lazy_matrix_mul(m_a, m_b)` using NumPy.

First, import the function:
    >>> lazy_matrix_mul = __import__('101-lazy_matrix_mul').lazy_matrix_mul

Test normal cases:
    >>> lazy_matrix_mul([[1, 2], [3, 4]], [[1, 2], [3, 4]])
    array([[ 7, 10],
           [15, 22]])

    >>> lazy_matrix_mul([[1, 2]], [[3, 4], [5, 6]])
    array([[13, 16]])
