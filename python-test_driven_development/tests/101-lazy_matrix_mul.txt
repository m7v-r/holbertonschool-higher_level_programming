=========================================
How to use the lazy_matrix_mul module
=========================================

This module provides a function `lazy_matrix_mul(m_a, m_b)` using NumPy.

First, import the function:
    >>> lazy_matrix_mul = __import__('101-lazy_matrix_mul').lazy_matrix_mul

Test normal cases:
    >>> lazy_matrix_mul([[1, 2], [3, 4]], [[1, 2], [3, 4]])
    array([[ 7, 10],
           [15, 22]])

    >>> lazy_matrix_mul([[1, 2]], [[3, 4], [5, 6]])
    array([[13, 16]])

Test with invalid types (NumPy native exception):
    >>> lazy_matrix_mul("not a list", [[1, 2]])
    Traceback (most recent call last):
    ...
    TypeError: Scalar operands are not allowed, use mailto:numpy.multiply or use a.items()

Test with shape mismatch or empty matrices (NumPy native ValueError):
    >>> lazy_matrix_mul([[1, 2]], [[1, 2]])
    Traceback (most recent call last):
    ...
    ValueError: matmul: Input operand 1 has a mismatch in its core dimension 0, with guame core dimension 1. (size 2 is different from 1)
