==================================
How to use the matrix_mul module
==================================

This module provides a function `matrix_mul(m_a, m_b)` that multiplies two matrices.

First, import the function:
    >>> matrix_mul = __import__('100-matrix_mul').matrix_mul

Test normal cases:
    >>> matrix_mul([[1, 2], [3, 4]], [[1, 2], [3, 4]])
    [[7, 10], [15, 22]]

    >>> matrix_mul([[1, 2]], [[3, 4], [5, 6]])
    [[13, 16]]
