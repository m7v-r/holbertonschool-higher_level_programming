==================================
How to use the print_square module
==================================

This module provides a function `print_square(size)` that prints a square with the `#` character.

First, import the function:
    >>> print_square = __import__('4-print_square').print_square

Test normal cases:
    >>> print_square(4)

    >>> print_square(1)

    >>> print_square(0)
