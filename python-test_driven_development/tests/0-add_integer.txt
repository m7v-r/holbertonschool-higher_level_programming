===============================
How to use the add_integer module
===============================

This module provides a function `add_integer(a, b=98)` that adds two numbers.
Both arguments must be integers or floats, and they are cast to integers before addition.

First, import the function:
    >>> add_integer = __import__('0-add_integer').add_integer

Test normal cases with integers:
    >>> add_integer(1, 2)
    3
    >>> add_integer(100, -2)
    98

Test using the default value for b (b=98):
    >>> add_integer(2)
    100

Test casting floats to integers:
    >>> add_integer(100.3, -2)
    98
    >>> add_integer(2.9, 2)
    4
